Project Showcase

1. Plant Disease Identification and Field Management System

This project focuses on developing a plant disease identification system using deep learning. The system enables farmers to upload images of their crops, and it identifies plant diseases based on visual symptoms using a convolutional neural network (CNN). After identifying a disease, the system suggests appropriate treatments and field management techniques. Real-time notifications are integrated through a Telegram bot, allowing users to receive urgent disease alerts and updates.

Key Features:

  • CNN-based disease detection from plant images.
  • Real-time notifications via Telegram for disease alerts.
  • Field management suggestions for disease prevention and control.

2. Mental Health Chatbot

This chatbot helps users identify potential mental health issues based on symptoms they provide. It utilizes a machine learning model trained to recognize six types of mental health concerns. The chatbot offers a conversational interface for input and diagnosis, along with appointment booking and user history tracking.

Key Features:

  • Machine learning-based diagnosis for six types of mental health issues.
  • Chat interface for symptom input and diagnosis output.
  • Appointment booking directly through the chatbot.
  • User history tracking to allow viewing of past diagnoses and interactions.

3. Medical Chatbot

The medical chatbot allows users to describe symptoms and receive potential diagnoses. It provides options for viewing recommended medications and precautions, along with an integrated appointment booking system and a history tracking feature.

Key Features:

  • Symptom-based diagnosis provided via the chat interface.
  • Details on precautions and medications available on user request.
  • Appointment booking system integrated within the chat.
  • History tracking allowing users to view past interactions and diagnoses.
